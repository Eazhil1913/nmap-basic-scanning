# Nmap Basic Scanning

## Description
Beginner level Nmap scanning practice project.

## Objective
To understand how port scanning works using Nmap.

## Tools Used
- Nmap
- Windows / Kali Linux

## Commands Practiced
- nmap localhost
- nmap -p 80 localhost
- nmap -sS localhost

## Learning Outcome
- Learned what open ports are
- Understood basic network scanning
